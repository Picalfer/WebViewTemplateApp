# WebViewTemplateApp

[![Kotlin](https://img.shields.io/badge/Kotlin-1.8+-blue.svg?logo=kotlin)](https://kotlinlang.org)
[![Platform](https://img.shields.io/badge/Platform-Android-green.svg?logo=android)](https://www.android.com/)

Шаблонное приложение с WebView, Firebase и системой уведомлений.

## О проекте

Приложение демонстрирует комплексное использование WebView с загрузкой контента из Firebase Firestore. Включает систему push-уведомлений и обработку состояний сети.

**Основной функционал:**
- WebView с сохранением состояния
- Загрузка URL из Firebase Firestore
- Push-уведомления через FCM
- Оффлайн-режим с кешированием
- Динамические разрешения для уведомлений

## Технологии

* Kotlin
* WebView
* Firebase (Firestore, Analytics, FCM)
* BroadcastReceiver
* Network monitoring
